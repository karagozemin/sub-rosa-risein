# Orange Belt demo video script

Target length: 3–5 minutes.

1. Show the GitHub repository name and explain that it is the independent
   **Sub Rosa Rise In Sprint**, not the production Sub Rosa frontend.
2. Open the README and briefly show White, Yellow, and Orange Belt evidence.
3. Show Freighter on **Testnet**, connect it, and point out the public key and
   live XLM balance.
4. Send a small testnet XLM payment, approve it in Freighter, and show the real
   transaction hash plus Stellar Expert link.
5. Show the deployed Soroban contract ID in the app.
6. Create a new round, approve the contract invocation, and show its transaction
   hash.
7. Enter a sample private allocation, explain that SHA-256 runs locally, submit
   the 32-byte commitment, and show the second contract transaction hash.
8. Refresh contract state and point out the synchronized round title, owner,
   creation time, and incremented commitment count.
9. Open the testnet explorer for one contract transaction.
10. Finish with pnpm test, pnpm contract:test, and the green GitHub Actions run.

Never show a secret key, recovery phrase, or real-value wallet in the recording.
